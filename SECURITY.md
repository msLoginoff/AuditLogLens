# Security Policy

## Supported Versions

AuditLogLens is currently in public alpha. Security fixes are expected to target the latest published package version.

| Version | Supported |
| ------- | --------- |
| Latest alpha | Yes |
| Older alpha versions | Best effort |

## Reporting a Vulnerability

Please do not open a public GitHub issue for a security vulnerability.

Report security issues privately through GitHub's private vulnerability reporting if it is available for this repository, or contact the maintainer directly through GitHub.

Include:

- Affected AuditLogLens version.
- A minimal description of the vulnerable scenario.
- Whether the issue can expose or incorrectly write audit data.
- Reproduction steps or a small proof of concept when possible.

The project is maintained as a small open-source library, so response times are best effort.
